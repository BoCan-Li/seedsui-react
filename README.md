## 简介
SeedsUI，专为移动设备设计的UI框架，组件全面、丰富、轻量、低耦合、侵入性小，目前有react版和vue版(h5版本可从react的commponets文件中直接抽取)

## 重要
* 环境必须支持less编译
* `seedsui-variables.less` 是换肤配置文件,可根据项目自行修改,升级时需要注意覆盖
